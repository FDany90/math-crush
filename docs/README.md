## Resumen ejecutivo (TL;DR)

- **Mercado puzzle:** +10.000M USD IAP en 2025 (+14% YoY), 2.º género que más crece. ✅
- **Plataformas:** Android = volumen barato (CPI $0,14-2); iOS = caro pero recupera 3× más (ROAS D30 47% vs 15%). ✅
- **Monetización:** híbrida (ads + IAP), ARPDAU $0,03-0,10, conversión a pago <5%. ✅
- **Marketing:** Meta + Apple Search Ads + TikTok. LatAm 15× más barato que EE.UU. (CPI $0,34 vs $5,28). 🟡
- **Estrategia:** Web como laboratorio → Android+LatAm para volumen → iOS para monetizar → híbrido desde día 1 → ads solo cuando LTV > CPI.
